## 🔧 P3 核心模块详情

### 1. 物体检测与分割 (segmentation.py)

```python
SemanticClass          # 语义类别枚举 (15+ 类)
├── 矿石: IRON_ORE, SILICON_ORE
├── 加工品: IRON_BAR, CIRCUIT_BOARD, MOTOR, JOINT_MODULE, ...
└── 设施: WORKSTATION, CHARGING_DOCK, ROBOT

DetectedObject         # 检测到的物体数据类
├── class_name: str    # 类别名称
├── confidence: float  # 置信度
├── bbox_2d: Tuple     # 2D 边界框
├── center_3d: ndarray # 3D 中心位置
├── estimated_size: ndarray # 估计尺寸
└── point_cloud: ndarray    # 物体点云

ObjectDetector         # 物体检测器
├── detect()           # 从 RGB-D + 分割图检测物体
└── 支持 3D 定位和尺寸估计

CameraIntrinsics       # 相机内参
├── from_fov()         # 从视场角创建
└── matrix             # 内参矩阵

add_realistic_noise()  # 噪声模拟 (sim-to-real)
├── RGB 高斯噪声
├── 深度噪声 + dropout
├── 亮度/对比度变化
└── 随机遮挡模拟
```

### 2. 位姿估计 (pose_estimation.py)

```python
ObjectPose             # 6D 位姿数据类
├── position: ndarray  # [x, y, z]
├── orientation: ndarray # [qx, qy, qz, qw]
├── rotation_matrix    # 旋转矩阵 (3x3)
├── transformation_matrix # 变换矩阵 (4x4)
├── euler_angles       # 欧拉角
└── yaw                # 偏航角

estimate_pose_pca()    # PCA 位姿估计
└── 从点云主成分分析估计朝向

estimate_pose_icp()    # ICP 位姿估计
├── 迭代最近点算法
├── 支持初始位姿估计
└── 返回位姿和误差

PoseEstimator          # 位姿估计器
├── estimate()         # 结合 PCA + ICP 精化
└── estimate_from_depth() # 从深度图估计

compute_pose_error()   # 计算位姿误差
└── 返回位置误差 (米) + 角度误差 (度)
```

### 3. 工站对接检测 (dock_detection.py)

```python
DockType               # 对接点类型枚举
├── INPUT_PORT         # 入料口
├── OUTPUT_PORT        # 出料口
├── CHARGING           # 充电口
└── STORAGE            # 存储槽

ArUcoMarker            # ArUco marker 信息
├── marker_id          # ID
├── corners            # 角点像素坐标
├── center             # 中心
└── pose_3d            # 3D 位姿

ArUcoDetector          # ArUco 检测器
├── 支持 OpenCV 4.7+ 新 API
├── 兼容旧版 API
└── 从深度图估计 3D 位姿

DockDetector           # 工站对接检测器
├── STATION_MARKER_MAP # 工站 marker ID 映射
├── detect_docks()     # 检测对接点
└── get_station_dock() # 获取指定工站对接点
```

### 4. 语义地图 (semantic_map.py)

```python
OccupancyStatus        # 占据状态枚举
├── UNKNOWN            # 未知
├── FREE               # 空闲
├── OCCUPIED           # 占据
└── OBSTACLE           # 障碍物

ZoneType               # 区域类型枚举
├── MINE_IRON, MINE_SILICON
├── PROCESSING, FABRICATION, ASSEMBLY
├── CHARGING, WAREHOUSE
└── PATH, SOLAR

SemanticMap            # 语义地图
├── resolution: 0.1m   # 栅格分辨率
├── width/height: 50m  # 地图尺寸
├── world_to_grid()    # 坐标转换
├── set_occupancy()    # 设置占据状态
├── set_semantic()     # 设置语义标签
├── update_from_depth() # 从深度图更新
├── add_item()         # 添加物品记录
├── get_nearest_item() # 获取最近物品
├── is_path_clear()    # 检查路径畅通
├── visualize()        # 生成可视化
└── save()/load()      # 持久化
```

### 5. 资源追踪 (resource_tracker.py)

```python
ZoneState              # 区域状态枚举
├── UNKNOWN, ACTIVE, DEPLETED
├── PROCESSING, IDLE, ERROR

ResourceState          # 资源状态
├── resource_type      # 资源类型
├── estimated_quantity # 估计数量
├── confidence         # 置信度
└── last_observed_time # 最后观察时间

StationState           # 工站状态
├── station_name, station_type
├── state              # 当前状态
├── input_buffer, output_buffer
├── current_recipe     # 当前配方
└── process_remaining  # 剩余处理时间

ResourceTracker        # 资源追踪器
├── update_mine_observation()    # 更新矿区观察
├── update_warehouse_item()      # 更新仓库物品
├── update_station_state()       # 更新工站状态
├── update_energy_state()        # 更新能量状态
├── decay_confidence()           # 置信度衰减
├── get_total_resources()        # 获取总资源
├── compare_with_ground_truth()  # 与真实状态比较
└── get_summary()                # 获取状态摘要
```

## 🔬 技术亮点

1. **模块化设计**: 5 个独立模块，职责清晰
2. **3D 感知**: 从 2D 图像反投影到 3D 世界坐标
3. **噪声鲁棒**: 支持 sim-to-real 噪声模拟
4. **位姿估计**: PCA + ICP 双算法支持
5. **语义地图**: 实时更新的 2D 栅格地图
6. **资源追踪**: 带置信度衰减的状态估计
7. **OpenCV 兼容**: 支持新旧 API 版本
