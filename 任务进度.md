### 🔧 核心模块详情

#### 1. Terrain (地形)
- 支持平坦/高度场/网格地形
- 摩擦系数材质配置
- 日照周期模拟

#### 2. MineZone (矿区)
- Poisson 圆盘采样生成矿石位置
- 支持铁矿/硅矿两种类型
- 矿石采集与状态管理

#### 3. SolarArray (太阳能阵列)
- 4 块太阳能板配置
- 日夜循环发电模拟
- 天气影响因子

#### 4. ChargingDock (充电站)
- 50W 充电功率
- 基于距离的对接检测
- 充电统计与状态管理

#### 5. PathNetwork (道路网络)
- 9 个路径节点
- A* 最短路径规划
- 支持区域到区域导航

#### 6. Warehouse (仓库)
- 24 个存储槽位 (4×3×2)
- 物品存取管理
- 库存统计

#### 7. Items (物品系统)
- 11 种物品类型定义
- 物品属性注册表
- 物品创建工厂

#### 8. Recipes (配方系统)
- 9 种制造配方
- 原材料需求计算
- 制造时间与能量成本

#### 9. RecipeGraph (依赖关系图)
- 自动构建 DAG
- 拓扑排序
- Mermaid/DOT 可视化

### 📊 制造需求分析

制造一个 `assembled_robot` 需要:

| 项目 | 数值 |
|------|------|
| 铁矿 (iron_ore) | 132 单位 |
| 硅矿 (silicon_ore) | 33 单位 |
| 总制造时间 | 72.5 分钟 |
| 总能量消耗 | 17.6 Wh |
| 制造步骤 | 27 步 |

**资源验证**: 世界配置中铁矿 500 单位, 硅矿 300 单位, 足够制造多个机器人。

### 🚀 使用方法

```bash
# 分析制造需求
python scripts/launch_world.py --analyze-requirements

# 启动世界 (运行 10 秒仿真)
python scripts/launch_world.py --simulate 10 --verbose

# 无头模式运行
python scripts/launch_world.py --headless
```
