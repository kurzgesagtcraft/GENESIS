## 🔬 技术亮点

1. **模块化设计**: 6 个控制模块 + 7 种技能，职责清晰
2. **Pure Pursuit**: 经典路径跟踪算法，支持自适应前视距离
3. **阻尼最小二乘法**: IK 求解器，处理奇异点

5. **阻抗控制**: 支持柔顺操作，用于精细装配
6. **异步技能**: 所有技能支持异步执行和取消
7. **完整测试**: 7 个测试模块全部通过

## 🔧 P6 核心模块详情

### 1. LLM 客户端 (llm_client.py)

```python
LLMProvider # LLM 提供商枚举
├── OPENAI # OpenAI GPT-4
├── VLLM # 本地 vLLM
├── OLLAMA # 本地 Ollama
└── MOCK # 测试用 Mock

LLMConfig # LLM 配置
├── provider # 提供商
├── model # 模型名称
├── api_key # API 密钥
├── base_url # 基础 URL
├── temperature # 温度参数
├── max_tokens # 最大 token 数
├── retry_count # 重试次数
└── enable_cache # 启用缓存

LLMClient # LLM 统一客户端
├── chat() # 发送聊天请求
├── chat_json() # 发送聊天请求并返回 JSON
├── get_stats() # 获取统计信息
└── clear_cache() # 清空缓存
```

### 2. 战略规划器 (strategic_planner.py)

```python
Task # 原子任务
├── task_id # 任务 ID
├── type # 任务类型
├── description # 描述
├── target # 目标物品
├── quantity # 数量
├── station # 目标工站
├── recipe # 目标配方
├── items # 携带物品
└── status # 状态

TaskPlan # 任务计划
├── goal # 目标
├── tasks # 任务列表
├── is_complete() # 是否完成
├── get_next_task() # 获取下一个任务
├── parse() # 解析 JSON
└── to_text() # 转换为文本

StrategicPlanner # 战略规划器
├── generate_master_plan() # 生成主计划
└── replan() # 重新规划
```

### 3. 行为树框架 (behavior_tree.py)

```python
NodeStatus # 节点状态
├── IDLE # 空闲
├── RUNNING # 运行中
├── SUCCESS # 成功
└── FAILURE # 失败

Blackboard # 黑板 (共享数据)
├── set() # 设置数据
├── get() # 获取数据
├── has() # 检查数据
└── clear() # 清空数据

TreeNode # 节点基类
├── tick() # 执行一个 tick
├── on_start() # 开始回调
├── update() # 更新逻辑
├── on_stop() # 停止回调
└── reset() # 重置状态

SequenceNode # 顺序节点
SelectorNode # 选择节点
ActionNode # 动作节点
ConditionNode # 条件节点
ParallelNode # 并行节点
RepeatNode # 重复节点

BehaviorTree # 行为树管理器
├── tick() # 执行一个 tick
└── reset() # 重置树
```

### 4. 任务执行器 (task_executor.py)

```python
TaskExecutor # 任务执行器
├── build_tree_from_plan() # 从计划构建行为树
├── _create_node_for_task() # 为任务创建节点
└── _execute_*_task() # 各类任务执行方法
```

### 5. 异常处理器 (error_handler.py)

```python
ErrorType # 错误类型
├── GRASP_FAILED # 抓取失败
├── NAVIGATION_BLOCKED # 导航受阻
├── STATION_ERROR # 工站错误
├── BATTERY_CRITICAL # 电量临界
├── ITEM_LOST # 物品丢失
└── UNKNOWN # 未知错误

Recovery # 恢复策略
├── actions # 恢复动作列表
├── max_retries # 最大重试次数
└── priority # 优先级

ErrorHandler # 异常处理器
├── handle_error() # 处理错误
├── request_replan() # 请求重规划
└── get_error_summary() # 获取错误统计
```

### 6. Dashboard (dashboard.py)

```python
DashboardState # Dashboard 状态快照
├── timestamp # 时间戳
├── robot_position # 机器人位置
├── robot_battery # 机器人电量
├── current_task # 当前任务
├── task_progress # 任务进度
├── inventory # 库存
├── station_status # 工站状态
├── energy_balance # 能量平衡
└── errors # 错误列表

Dashboard # 实时监控
├── update() # 更新状态
├── get_current_state() # 获取当前状态
├── get_summary() # 获取摘要
├── export_history() # 导出历史
└── render_text() # 渲染文本
```

## 🔬 技术亮点

1. **多后端 LLM 支持**: OpenAI、vLLM、Ollama、Mock 四种后端
2. **智能缓存**: LLM 响应缓存，减少重复调用
3. **重试机制**: 带退避的重试逻辑，提高稳定性
4. **行为树框架**: 完整的 BT 节点类型，支持复杂任务编排
