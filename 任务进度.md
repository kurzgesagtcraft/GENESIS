## 🔧 P2 核心模块详情

### 1. 机器人规格说明书 (docs/robot_spec.md)

完整的 GENESIS-Bot v1 规格定义：

| 组件 | 规格 |
|------|------|
| **底盘** | 四轮差速, 0.6m × 0.4m × 0.15m, 最大速度 1.5 m/s |
| **躯干** | 圆柱形, 高度 0.8m, 半径 0.1m |
| **手臂** | 双 6-DOF, 臂展 0.7m, 末端负载 5kg |
| **夹爪** | 平行两指, 开合范围 0~0.1m, 夹持力 0~50N |
| **传感器** | RGB-D 相机×3, IMU×1, 力/力矩传感器×2 |
| **电池** | 500 Wh, 待机 10W, 移动 50W, 操作 80W |

### 2. URDF 模型

| 文件 | 描述 | 关节数 |
|------|------|--------|
| chassis.urdf | 四轮差速底盘 | 4 (continuous) |
| arm.urdf | 6-DOF 串联臂 | 6 (revolute) |
| gripper.urdf | 平行两指夹爪 | 2 (prismatic) |
| genesis_bot.urdf | 完整机器人 | 18 (含夹爪) |

### 3. 传感器系统 (sensors.py)

```
SensorBase          # 传感器基类
├── RGBDCamera      # RGB-D 深度相机
│   ├── 分辨率: 640×480 (头部) / 320×240 (腕部)
│   ├── FOV: 70° (头部) / 90° (腕部)
│   └── 输出: RGB + Depth + PointCloud
├── IMUSensor       # 惯性测量单元
│   ├── 更新率: 200 Hz
│   └── 输出: 角速度 + 线加速度 + 姿态
└── FTSensor        # 六维力/力矩传感器
    ├── 更新率: 1000 Hz
    └── 输出: 力 + 力矩 (6维)

SensorSuite         # 传感器套件管理器
```

### 4. 电池系统 (battery.py)

```
Battery             # 电池类
├── 容量: 500 Wh
├── SOC 管理: 0-1 范围
├── 功耗模式:
│   ├── IDLE: 10 W
│   ├── MOBILE: 50 W
│   ├── MANIPULATION: 80 W
│   └── PERCEPTION: 20 W
├── 充电效率: 90%
└── 临界电量: 15%

PowerMode           # 功耗模式枚举
```

### 5. 机器人控制接口 (robot_interface.py)

```
GenesisBot          # 机器人统一控制接口
├── 底盘控制
│   ├── move_to(x, y, yaw)      # 导航到目标位置
│   ├── set_velocity(linear, angular)  # 设置速度
│   └── get_base_pose()         # 获取位姿
├── 手臂控制
│   ├── move_arm_to_joints(arm, positions)  # 关节空间运动
│   ├── move_arm_to_pose(arm, pose)         # 笛卡尔空间运动
│   └── get_arm_state(arm)      # 获取手臂状态
├── 夹爪控制
│   ├── grasp(arm, width, force)  # 抓取
│   ├── release(arm)              # 释放
│   └── get_gripper_state(arm)    # 获取夹爪状态
├── 感知
│   ├── get_head_image()         # 获取头部相机图像
│   ├── get_wrist_image(arm)     # 获取腕部相机图像
│   └── get_wrist_force(arm)     # 获取力/力矩数据
└── 状态
    ├── get_battery_soc()        # 获取电池电量
    └── get_full_state()         # 获取完整状态
```
