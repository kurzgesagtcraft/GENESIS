#### 技术选型决策

| 决策项 | 选择 | 置信度 |
|--------|------|--------|
| 仿真引擎 | Isaac Sim 4.x (主) + MuJoCo (辅) | 0.90 |
| RL 框架 | Isaac Lab | 0.85 |
| 策略架构 | ACT (LeRobot) | 0.80 |
| 机器人模型 | MJCF 格式 | 0.90 |
| 手臂选型 | Franka Panda | 0.85 |
| LLM 规划 | Code-as-Policies | 0.75 |

#### 风险点识别

- **R1**: Isaac Sim 学习曲线陡峭 → 先用 MuJoCo 原型
- **R2**: 硬件要求高 → 云 GPU 备选
- **R3**: LLM 规划实时性不足 → 缓存 + 预计算
- **R4**: Sim-to-Real gap → 域随机化
