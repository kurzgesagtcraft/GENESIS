# GENESIS-Bot v1 机器人规格说明书

## 一、总体概述

GENESIS-Bot v1 是一款移动操作机器人，专为自复制最小闭环模拟系统设计。该机器人具备以下核心能力：

- **自主导航**：四轮差速底盘，支持在工厂环境内自主移动
- **双臂操作**：双 6-DOF 机械臂，可协同完成复杂装配任务
- **环境感知**：多传感器融合，包括 RGB-D 相机、IMU、力/力矩传感器
- **能源管理**：电池供电，支持自主充电

### 2.3 差速驱动运动学

```
轮子速度计算:
  v_left = v_linear - ω_angular × (wheel_base / 2)
  v_right = v_linear + ω_angular × (wheel_base / 2)

其中:
  v_linear: 线速度 (m/s)
  ω_angular: 角速度 (rad/s)
  wheel_base: 轮距 (m)
```
